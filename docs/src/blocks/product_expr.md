# Product Expressions

Used to create `*`, `/` or `%` arithmetic expression.

For example, it can be used like this:

![Example of comparison](../images/multiplicative_op_example.png)

and would produce the following Essence Output:

```essence

```

See [here](https://conjure.readthedocs.io/en/latest/essence.html#arithmetic-operators) for further information surrounding arithmetic operators.
