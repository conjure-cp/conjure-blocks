Instructions

  1. Drag blocks into Block Essence panel to build constraint program.
    1. Click the cogs to add more inputs
    2. Click the question marks to see helpful comments about block input
    3. Hover on the blocks to see tooltips linked to documentation for that block
  2. Textual Essence is generated automatically in Generated Essence Code panel.
  3. Specify value for parameters in JSON blocks in Block Data Input panel.
  4. Solver constaint problem with SOLVE button, JSON solution displayed beneath JSON solution panel, and block solution in Block Solution.
  5. Download .essence file with DOWNLOAD button.
